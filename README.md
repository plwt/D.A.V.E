# VoiceAssist


### Requirements

```sudo apt-get install portaudio19-dev```

```sudo apt-get install python3-tk python3-dev```

```sudo apt update && sudo apt install espeak ffmpeg libespeak1```

### Commands

- Open notepad
- Open Firefox
- Open Thunderbird
- Open files
- Open Zoom
- Open calculator
- Open YouTube 
- Search on Google
- End session
- What is the news? 
- What is the weather?
- Open terminal
